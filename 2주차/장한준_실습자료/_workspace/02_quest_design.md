# 02. NPC 호감도 퀘스트 설계

## 1. 설계 철학

Project PA의 NPC 퀘스트는 **"심부름을 가장한 고백"** 을 지향한다. 표면적으로는 아이템 조달(Fetch)이지만, 각 퀘스트는 NPC의 상처·세계관의 진실·설계자의 정체성 중 최소 한 축을 드러내는 장치여야 한다. T0 경계 단계의 퀘스트는 "기술적 실용성"만으로 위장되지만, T4 결속 단계로 갈수록 아이템 자체가 상징(유품, 증거, 선택의 징표)이 되며 플레이어는 자신이 '유기된 설계자'임을 NPC에게 밝힐지 결정하게 된다. 모든 아이템은 `01_worldbuilding.md`의 현존 게임 시스템 테이블을 재사용해 개발 작업량을 최소화한다.

## 2. 공통 규칙

### 2.1. 호감도 보상 기준선

| 티어 | 해금 트리거 | 기본 보상 | 분기 편차 | 실패 페널티 |
|---|---|---|---|---|
| T0→T1 | 첫 심부름 (공통 메타 테마: "신뢰의 첫 단추") | +15 | ±0 | -5 |
| T1→T2 | 거점 인근 원정·시간제한 | +20 | ±5 | -10 |
| T2→T3 | 위험 지역·도덕 선택 포함 | +25 | ±10 | -15, 재도전 1회 제한 |
| T3→T4 | 개인사 상징 아이템 | +35 | ±10 | -25, NPC 영구 이탈 가능 |

### 2.2. Fetch 하위 변형 (Quest Design Patterns 아키타입 #5 수집을 중심으로 변주)

- **Gather**: 특정 자원을 누적 수집 (단순·T1 위주)
- **Retrieve**: 지정된 단일 유물 회수 (위험도 중·T2 위주)
- **Escort-Delivery**: 아이템과 함께 NPC 또는 취약 대상 이동 (T2~T3)
- **Trade-Fetch**: 타 상인/세력과 교환해 조달 (도슨 전담·T2~T3)
- **Moral Fetch**: 획득한 아이템의 최종 귀속처를 플레이어가 선택 (T3~T4 분기 핵심)

### 2.3. 실패 정의

- 시간 제한 만료, 아이템 품질 미달, 분기 선택 실패, NPC 사망(호위 대상) 중 하나 이상이 명시될 때만 "실패"로 판정.
- T0~T2 퀘스트는 재도전 가능. T3부터는 호감도 페널티와 함께 재도전 1회 제한. T4는 1회성 분기.

### 2.4. 퀘스트 아키타입 매핑 (공통)

모든 16개 퀘스트는 "수집(#5)"을 표면층으로 하되, 기계층에서 아래 아키타입 중 1개 이상을 결합한다: #4 탐험 / #7 호위 / #8 협상 / #9 배달 / #12 딜레마.

## 3. NPC별 퀘스트 상세

### 3.1. 리오 (Leo) — 정수 시스템 전문가

**색채 요약**: 정수·담수 아이템 중심. 철학적·기술적 질문이 선택지에 섞인다. 후반은 "설계자가 엑소더스 관련자임을 스스로 밝히는가"가 핵심.

---

#### Q_LEO_01 — 멈춰버린 필터

- **의뢰 NPC / 해금 호감도 티어**: 리오 / T0 → T1
- **퀘스트 타입**: Gather (단순 수집) — 공통 메타 "신뢰의 첫 단추"
- **목표 아이템**: 역삼투막 필터 2개 (정수/담수화 카테고리), 염도 측정기 1개
- **목표 장소**: 거점 인근 해안 잔해장 (부유 거점 기준 500m 이내 수상 잔해)
- **단계**:
  1. 리오의 작업장에서 정지된 개인 필터 장비 확인 — 목표: 상호작용 / 힌트: 리오가 지도에 잔해장 위치 표시
  2. 잔해장에서 역삼투막 필터 2개 수거 — 목표: 수집 / 힌트: 상자형 해양 잔해를 수색
  3. 염도 측정기 확보 — 목표: 수집 / 선택지: 잔해 속에서 2개 발견 시 "여분 1개를 개인 보유" vs "전부 리오에게 전달"
  4. 리오에게 귀환 — 목표: 전달
- **선행 조건**: 없음 (T0 시작 시 자동 해금)
- **실패 조건**: 없음 (재도전 무제한)
- **성공 보상**:
  - 호감도 +15 (전량 전달 시 +20)
  - 정화수 5병 (즉시 소비 가능)
  - 거점 정수기 가동 조건 1단계 해금
- **실패 페널티**: -5 (72시간 이상 방치 시)
- **분기 포인트**: `flag_leo_trust_seed` — "전량 전달"이면 TRUE, "여분 보유"면 FALSE. 이후 Q_LEO_03에서 리오의 태도 차이로 반영.
- **연관 세계관 요소**: 리오의 약점 — "기술 부품 부족으로 개인 필터 장비가 멈춰 있음"

---

#### Q_LEO_02 — 염해(鹽害)의 경계

- **의뢰 NPC / 해금 호감도 티어**: 리오 / T1 → T2
- **퀘스트 타입**: Retrieve (원정 회수) + 시간제한
- **목표 아이템**: 구형 변압기 1개 (거점 에너지 카테고리), 정수 튜브 3개
- **목표 장소**: 남서쪽 잔존 고지대 — 침수된 과거 정수 시설 "해수 담수 3호기"
- **단계**:
  1. 리오로부터 해수 담수 3호기 좌표와 출입 코드 수령 — 목표: 대화
  2. 만조 전 48시간 이내 시설 도달 — 목표: 원정 / 힌트: 기상 예보 확인
  3. 변압기와 정수 튜브 회수 — 목표: 수집 / 환경 이벤트: 부식된 바닥, 짧은 탐색 구간
  4. 만조가 오기 전 귀환 — 목표: 시간 내 복귀
- **선행 조건**: Q_LEO_01 완료, 호감도 T1 이상
- **실패 조건**: 48시간 초과 시 시설 수몰, 아이템 영구 소실
- **성공 보상**:
  - 호감도 +20
  - 구형 변압기 (거점 전력망 업그레이드 가능)
  - 리오의 "해수 담수화 초기 도면 파편" 1장
- **실패 페널티**: -10, 재도전 가능하나 아이템 난이도 상승(더 깊은 수심)
- **분기 포인트**: 없음 (단, 정수 튜브를 3개가 아닌 2개만 가져올 경우 리오가 "충분하다"고 넘어가며 `flag_leo_pragmatic` SET)
- **연관 세계관 요소**: 리오의 거점 가치 — "해수 담수화 장비 수리·개량"

---

#### Q_LEO_03 — 오염된 우물

- **의뢰 NPC / 해금 호감도 티어**: 리오 / T2 → T3
- **퀘스트 타입**: Moral Fetch (도덕적 조달 분기)
- **목표 아이템**: 정수 튜브 5개 (완제품), 역삼투막 필터 1개 (고품질)
- **목표 장소**: 북쪽 수몰지 외곽 — 소규모 난민 집단 "표류민 선단"
- **단계**:
  1. 리오가 인근 난민 집단의 식수원이 오염되었다는 소식을 공유 — 목표: 대화
  2. 플레이어가 직접 정수 튜브 5개를 제작·확보 — 목표: 제작/수집
  3. 난민 집단 방문 — 목표: 이동 / 이벤트: 난민 대표가 "필터 기술 자체"를 요구
  4. **분기 선택**:
     - (A) 완제품 전달만 하고 귀환 — 난민 일시 구제
     - (B) 리오의 기술 일부를 난민에게 가르쳐주도록 리오를 설득 — 장기 해결, 거점 자원 유출
     - (C) 물자를 거점 비축으로 돌림 — 난민 방치
- **선행 조건**: Q_LEO_02 완료, 호감도 T2 이상
- **실패 조건**: (C) 선택 후 난민 집단 전멸 이벤트 발생 시 자동 실패 처리
- **성공 보상**:
  - (A) 호감도 +25, 고품질 필터 제작 도면
  - (B) 호감도 +35, 리오의 "공개 신뢰" 대사 해금, 타 거점 교역 루트 오픈
  - (C) 호감도 -15, 리오의 실망 대사, Q_LEO_04 진입 난이도 상승
- **실패 페널티**: -15, 재도전 1회 제한
- **분기 포인트**:
  - `flag_leo_philanthropy = A|B|C`
  - (B) 선택 시 `flag_dawson_rumor_leo` SET — 도슨의 정보망에 리오의 개방성이 소문으로 흘러감
- **연관 세계관 요소**: "공동체의 규모가 곧 생존의 확률" 테마의 직접적 구현

---

#### Q_LEO_04 — 교신 기록 (T4 결속 엔딩 퀘스트)

- **의뢰 NPC / 해금 호감도 티어**: 리오 / T3 → T4
- **퀘스트 타입**: Retrieve + Moral Fetch (자기 고백 분기)
- **목표 아이템**: 설계도 파편 3장 (제작·가구 카테고리 — 엑소더스 교신 장비 설계도), 배터리 코어 1개
- **목표 장소**: 대범람 이전 수도권 외곽 — 수몰된 엑소더스 지상 통신소 잔해
- **단계**:
  1. 리오가 "당신에게 묻고 싶은 것이 있다"며 통신소 탐사를 제안 — 목표: 대화
  2. 통신소 잠수 탐사 (위험 지역) — 목표: 원정 / 환경: 저산소, 장비 필수
  3. 설계도 파편 3장과 배터리 코어 회수 — 목표: 수집 / 이벤트: 파편 중 1장에 설계자의 과거 서명 존재
  4. 귀환 후 리오가 서명을 들이대며 직접 질문 — **분기 선택**:
     - (A) "내가 엑소더스의 설계자였다"고 인정
     - (B) "다른 사람의 유품일 뿐"이라고 부정
     - (C) 침묵 (대화창 이탈)
- **선행 조건**: Q_LEO_03 완료 (A 또는 B), 호감도 T3 이상
- **실패 조건**: 통신소 탐사 중 장비 미비로 사망 → 재도전 불가, 리오의 결속 엔딩 영구 차단
- **성공 보상**:
  - (A) 호감도 +45, **리오 영입 확정**, 거점 냉각수 공급 시스템 해금, 전용 대사 "나는 너를 용서하지 않지만, 함께 살아남겠다" 해금
  - (B) 호감도 +25, 리오는 거점에 머물되 심층 대화 영구 차단, 리오의 기술 고급 단계 일부 잠김
  - (C) 호감도 -25, 리오 영구 이탈
- **실패 페널티**: -25, NPC 영구 이탈 가능
- **분기 포인트**:
  - `flag_leo_ending = BONDED_TRUTH | COEXIST_LIE | LOST`
  - `flag_designer_identity_revealed_to_leo` — (A)에서 TRUE, 이후 한(Han) 퀘스트 분기에도 영향
- **연관 세계관 요소**: "유기된 설계자" 영입 논리의 첫 시험 — *"우리가 버려진 것은 사실이지만, 그들이 남긴 유산으로 살아남아야 한다"*

### 3.2. 사라 (Sara) — 기상 제어기 수리공

**색채 요약**: 어린 동생 보호가 동기. 시간 제한(폭풍 도달 전)이 설계의 상수. 후반은 "위험한 원정에 사라를 데려갈지"가 분기.

---

#### Q_SAR_01 — 젖은 풍속계

- **의뢰 NPC / 해금 호감도 티어**: 사라 / T0 → T1
- **퀘스트 타입**: Gather (단순 수집) — 공통 메타 "신뢰의 첫 단추"
- **목표 아이템**: 풍속계 부품 3개 (기상 차폐막 카테고리), 배선 키트 1개
- **목표 장소**: 거점 옥상 및 주변 잔교 (거점 내부)
- **단계**:
  1. 사라의 동생이 장난감처럼 다뤄 분해해버린 풍속계를 확인 — 목표: 상호작용
  2. 거점 내 분산된 부품 3개 수거 — 목표: 수집 / 힌트: 동생과 짧은 대화 시 위치 힌트
  3. 배선 키트 확보 (거점 작업대) — 목표: 제작/수집
  4. 사라에게 전달 — 목표: 전달
- **선행 조건**: 없음 (T0 자동 해금)
- **실패 조건**: 없음
- **성공 보상**:
  - 호감도 +15
  - 기상 예보기 24시간 조기 경보 기능 해금 (거점 차폐막 연계)
- **실패 페널티**: -5 (72시간 방치 시)
- **분기 포인트**: `flag_sara_sibling_met` SET — 동생과 대화 이벤트 발생 여부
- **연관 세계관 요소**: 사라의 거점 가치 — "기상 차폐막 정비, 폭풍 조기 경보"

---

#### Q_SAR_02 — 폭풍 전 3시간

- **의뢰 NPC / 해금 호감도 티어**: 사라 / T1 → T2
- **퀘스트 타입**: Retrieve + 강한 시간 제한
- **목표 아이템**: 제어 모듈 1개 (기상 차폐막 카테고리), 방풍 합금판 2개
- **목표 장소**: 북동쪽 수몰지 — 구형 기상 관측소 잔해
- **단계**:
  1. 사라가 기상 예보기로 3시간 후 폭풍 도달을 감지 — 목표: 대화 / 긴급 트리거
  2. 관측소로 원정, 제어 모듈 회수 — 목표: 수집
  3. 방풍 합금판 2개 해체 — 목표: 수집 / 환경: 구조물 붕괴 리스크
  4. 폭풍 도달 전 거점 귀환 — 목표: 시간 내 복귀
- **선행 조건**: Q_SAR_01 완료, 호감도 T1 이상
- **실패 조건**: 3시간 초과 시 폭풍이 거점 타격 — 거점 HP 일부 손실, 아이템 파손
- **성공 보상**:
  - 호감도 +20 (시간 여유 30분 이상 남기면 +25)
  - 제어 모듈 (거점 차폐막 가동 준비 단계 해금)
  - 사라의 "실용주의자 인정" 대사
- **실패 페널티**: -10, 거점 자재 손실, 재도전 가능
- **분기 포인트**: `flag_sara_competence_proof` — 시간 내 여유 30분 이상 TRUE
- **연관 세계관 요소**: "기상은 통제 불능: 개인 장비로는 대응 불가" — 차폐막 필요성의 구체화

---

#### Q_SAR_03 — 동생의 장난감

- **의뢰 NPC / 해금 호감도 티어**: 사라 / T2 → T3
- **퀘스트 타입**: Moral Fetch (도덕 선택)

- **목표 아이템**: 도면 사본 1장 (제작·가구 카테고리) — 플레이어가 제공 / 대가로 도슨이 배터리 코어 1개 제공
- **목표 장소**: 거점 선착장 — 도슨의 부유 상선

  3. 도슨에게 전달 — 목표: 전달
  4. 도슨이 "이 정도면 서로 믿을만하다"며 배터리 코어 1개를 건넴 — 목표: 수령
- **선행 조건**: 없음
- **실패 조건**: 고등급 도면(예: 리오의 정수 도면) 제공 시 도슨이 즉시 수락하나 거점 자원 유출, `flag_dawson_exploit` SET
- **성공 보상**:
  - 호감도 +15
  - 배터리 코어 1개
  - 도슨 상점 기본 목록 해금
- **실패 페널티**: -5 (거래 취소 시)
- **분기 포인트**: `flag_dawson_first_trade = FAIR | EXPLOITED` — (고등급 도면 제공 시 EXPLOITED, 이후 도슨의 플레이어 평가 "호구" 성향)
- **연관 세계관 요소**: 도슨의 초기 태도 — "거래로만 관계를 맺는다"

#### Q_DAW_03 — 세 방향의 줄다리기

- **의뢰 NPC / 해금 호감도 티어**: 도슨 / T2 → T3
- **퀘스트 타입**: Moral Fetch (삼자 이해관계 분기)
- **목표 아이템**: 과급기 코어 1개 — 한 개의 아이템, 세 수요자
- **목표 장소**: 남동쪽 수몰지 — 좌초된 대형 쾌속선 "귀환의 약속"
- **단계**:
  1. 도슨이 "쾌속선에 과급기 코어 하나가 남아있다. 그걸 누구에게 줄지는 너에게 맡기겠다"며 경쟁 상황을 알림 — 목표: 대화
  2. 쾌속선 원정, 과급기 코어 회수 — 목표: 수집 / 환경: 경쟁 상인 NPC "베야"가 동시에 도달
  3. 회수 과정에서 베야와 조우 — 목표: 협상/전투 / 처리 방식 분기
  4. **분기 선택**: 누구에게 코어를 줄 것인가?
     - (A) **거점(자기 소유)** — 한의 Q_HAN_04 난이도 대폭 완화
     - (B) **도슨(빚 상환용)** — 도슨의 빚쟁이 정리, 도슨이 거점에 더 오래 머물 결심
     - (C) **베야(경쟁 상인)** — 보상 즉시 현금화, 그러나 도슨 격노
     - (D) **쾌속선 원 주인(숨겨진 의뢰인 — C 거점 소문과 연결)** — 과거 도슨이 붕괴시킨 공동체의 생존자, 도슨에게 정면 타격
- **선행 조건**: Q_DAW_02 완료, 호감도 T2 이상
- **실패 조건**: 베야와의 협상/전투 실패 시 코어 탈취, 퀘스트 실패 / (D) 진입은 `flag_dawson_rumor_heard` TRUE가 필수
- **성공 보상**:
  - (A) 호감도 +20, 거점 엔진 계통 업그레이드
  - (B) 호감도 +35, 도슨 정착 의사 강화 (`flag_dawson_settle_intent` SET)
  - (C) 호감도 -15, 도슨 상점 가격 +30% 페널티
  - (D) 호감도 -25 (도슨) / +외부 숨겨진 거점 신뢰 +20, Q_DAW_04 분기 "속죄" 루트 해금
- **실패 페널티**: -15, 재도전 1회 제한
- **분기 포인트**:
  - `flag_dawson_triangle = A|B|C|D`
  - (D) 선택은 Q_DAW_04 엔딩의 "속죄" 루트 전제 조건
  - (C) 선택은 `flag_dawson_hostile_merchant` SET — 도슨이 거점에 머무르지 않음
- **연관 세계관 요소**: 도슨의 핵심 상처 — "전 공동체가 내부 분열로 붕괴"의 반복 또는 단절

#### Q_DAW_04 — 정착의 값 (T4 결속 엔딩 퀘스트)

- **의뢰 NPC / 해금 호감도 티어**: 도슨 / T3 → T4
- **퀘스트 타입**: Escort-Delivery + Moral Fetch
- **목표 아이템**: 도슨의 이동 상선에 설치될 "정착 가구" 세트 — 구형 변압기 2개, 방풍 합금판 3개, 배선 키트 4개, 도면 사본 2장
- **목표 장소**: 거점 선착장 + 도슨의 상선 내부 + (D 루트의 경우) 과거 붕괴된 공동체 유적
- **단계**:
  1. 도슨이 "이 거점에 상선을 영구 정박시킬지, 아니면 마지막 한 번 더 떠날지 결정해야 한다"며 도움을 요청 — 목표: 대화
  2. 정착 가구 제작·조달 — 목표: 제작/수집
  3. **분기에 따른 차등 진행**:
     - (A/B 루트) 도슨의 상선에 가구를 설치하며 정박 준비 — 목표: 전달
     - (D 루트 — 속죄) 도슨을 호위해 과거 붕괴시킨 공동체 유적을 방문, 유품을 안치 — 목표: 호위·배달
     - (C 루트) 도슨은 거점을 떠나려 함. 마지막 거래를 할지 결정 — 목표: 협상
  4. **최종 분기 선택**:
     - (가) 정박 승인·영입 확정 (A/B/D 루트만 가능)
     - (나) 정박 거부 — 도슨은 떠나되 우호 거래 상인으로 남음
     - (다) 도슨을 추방 — 도슨의 정보가 적대 세력으로 유출 가능성
- **선행 조건**: Q_DAW_03 완료, 호감도 T3 이상
- **실패 조건**: (D 루트 속죄) 호위 중 유적에서 생존자들이 도슨을 공격, 구출 실패 시 도슨 사망 → 영구 이탈
- **성공 보상**:
  - (A/B 루트 + 가) 호감도 +35, **도슨 영입 확정**, 거점 상설 상점 해금, 타 거점 정보 맵 전체 공개, "여기서 죽는 것도 나쁘지 않겠다" 엔딩 대사
  - (D 루트 + 가) 호감도 +45, 도슨 영입 확정, "속죄의 상인" 타이틀 해금, 붕괴된 공동체 생존자 소집 이벤트 추가 해금
  - (나) 호감도 +15, 도슨 이동 상인 상태 유지 (상점 순회 방문)
  - (다) 호감도 -35, 도슨 적대화, 적대 상인 세력 등장
- **실패 페널티**: -25, (D 루트 호위 실패)의 경우 NPC 영구 이탈
- **분기 포인트**:
  - `flag_dawson_ending = BONDED_SETTLE | BONDED_ATONE | WANDERER | EXILED`
  - (D 루트 + 가) 달성 시 `flag_world_community_revival` SET — 거점 외부 세력 이벤트 체인 해금
- **연관 세계관 요소**: "공동체의 규모가 곧 생존의 확률" — 배신당해 도망친 자조차 공동체로 돌아올 수 있는가
